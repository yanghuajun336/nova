# NOVA

NOVA is an enterprise-oriented DevOps platform design project focused on building a standardized, platformized CI/CD system around Gitea, Jira, Harbor, Kubernetes, SonarQube, and related tooling.

## Current status

This repository is currently in the architecture and design phase.

## Goals

- Platformized CI/CD for multi-language microservices
- Standardized pipeline governance
- Jira-linked end-to-end delivery workflow
- Automated deployment, approval, rollback, and reporting
- Future AI-agent friendly CLI integration
